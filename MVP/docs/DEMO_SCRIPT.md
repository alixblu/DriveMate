# DriveMate AI 3-Minute Demo Script

## Open
- “VETC is a strong toll product, but it is still a low-frequency utility.”
- “DriveMate AI turns it into a daily mobility assistant by helping drivers save money and avoid waste before every trip.”

## Scene 1: Hero State
- Open `/?tab=home&scenario=fuel-hike`
- Say:
  “Tomorrow fuel rises by 500 VND per litre. Instead of showing generic news, DriveMate calculates what that means for me personally.”
- Point to:
  - Daily Decision Card
  - leave time
  - best-value route
  - refuel advice
  - expected weekly savings

## Scene 2: Route Comparison
- Open `/?tab=routes&scenario=fuel-hike`
- Say:
  “We do not just optimize for time. We compare exactly three route choices by ETA, toll, fuel, and total trip cost.”
- Point to:
  - `Fastest`
  - `Cheapest`
  - `Best Value`

## Scene 3: AI Explanation
- Open `/?tab=assistant&scenario=fuel-hike`
- Say:
  “Qwen on Alibaba Cloud explains the recommendation in natural language from structured forecast and routing payloads.”
- Use quick prompts:
  - `Why refuel tonight?`
  - `Best route today?`

## Scene 4: Support Features
- Open `/?tab=wallet&scenario=fuel-hike`
- Say:
  “Wallet and rewards are still valuable, but now they are downstream of the main recommendation instead of being separate product islands.”

## Scene 5: Reliability Backup
- Open `/?tab=home&scenario=fallback`
- Say:
  “If live forecasting fails during the demo, the app falls back to seeded outputs and keeps the same user story intact.”

## Close
- “DriveMate AI helps Vietnamese drivers decide when to leave, which route to take, and when to refuel to minimize total trip cost.”
- “That gives VETC a real path from utility to habit product.”
